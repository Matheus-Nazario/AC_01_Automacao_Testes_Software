# ⛏  Tech & Dependências

### Pré-requisitos

* Python 3.10.11
* pytest 7.4.0

<hr>

## Clonando o serviço

```sh
git clone https://github.com/Matheus-Nazario/AC_01_Automacao_Testes_Software.git
 
```

# Executando o serviço

Na pasta raiz do projeto executar o comando:

```sh
  pytest test/test_Ac01Exercicios.py -v
```
